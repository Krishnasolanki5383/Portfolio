# Krishna Solanki - Personal Portfolio ✨

A modern, interactive, and visually engaging personal portfolio built with **React** and **Vite**. Showcasing my projects, hackathon journeys, achievements, and skills through a responsive and seamless user experience.

## ✨ Features

- **Modern UI & Animations:** Smooth scrolling and interactive elements without relying heavily on external libraries.
- **Dynamic Projects Section:** Detailed showcase of cloned/built projects (PRACTO, HARIO, LIQUID DEATH, WIX) with custom images and scroll animations.
- **Interactive "Journey" Section:** A dedicated, minimalist Hackathon space featuring floating photo cards that expand on hover.
- **Certifications & Achievements:** Highlighting significant milestones, including rank achievements.
- **Social Integrations:** Easily accessible links to YouTube, X (Twitter), GitHub, and LinkedIn.
- **Responsive Design:** Fully optimized for both desktop and mobile viewing.
- **Preloader:** Custom initial loading experience.

## 💻 Running Locally

Follow these steps to run the portfolio on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Krishnasolanki5383/Portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd MY-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
